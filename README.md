# Battery prophet
Package that encourages best battery practices for your laptop

### Getting started
#### Prerequisites
- Git
- Yarn/NPM
#### Installations
- Clone the repo
```js
git clone git@github.com:M-ZubairAhmed/battery-prophet.git
```
- Install dependencies
```js
yarn
```
- Run program
```js
yarn start
```
- Open website
```
http://localhost:9091/
```
